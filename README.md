## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ git clone git@github.com:asnikitin/optimacros-back.git
$ cd optimacros-back
$ npm install
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

List the cars sorted by `name`

```bash
$ curl "http://localhost:3000/cars?sort=name"
```
